AUTOMATIC1111

El software que yo utilizo es AUTOMATIC1111, una herramienta ampliamente reconocida por ser intuitiva, fácil de usar y altamente compatible con una variedad de modelos de inteligencia artificial. Su interfaz flexible permite personalizar parámetros de generación de imágenes, lo que lo convierte en una excelente opción tanto para principiantes como para usuarios avanzados.

Requisitos

Hardware

GPU NVIDIA con al menos 4GB de VRAM (se recomiendan 8GB o más para un rendimiento óptimo). Sistema operativo Windows 10/11, Linux o macOS. Al menos 10GB de espacio libre en disco.

Software

Python 3.10.x y Git

Pasos instalacion

1. Instalar python 3.10

2. Instalar git

3. Clonar el repositorio de AUTOMATIC1111

4. Descargar un modelo de Stable Diffusion y meterlo en stable-diffusion-webui/models/Stable-diffusion

5. Instalar dependencias haciendo cd stable-diffusion-webui, y ejecutando webui.user.bat

6. Entrar en webui.user.bat y empezar a generar imagenes

Para mas informacion de que parametros usar y como empezar a generar imagenes visitar el siguiente video
